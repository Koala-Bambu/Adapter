# Adapter
simple pygame (game) code in which you must code an AI to defeat another adaptable AI, you only get three members while they get 5 think this through

# README
have you ever plan a raid, I hope not, but anyways, in this simple game you must create a specific ruleset to take over the ~~Weed~~ farm.

# CONTENTS
Two adaptive _*AI's*_ with different aproaches.
### MAFIA AI
* Your number one adversary                                               
* It will adapt slower but it has no upper limit                          
* will truly test your strategy                                           

### LAW AI
- The annoyance                                                           
- It adapts quickly but it cant adapt to everything                       
- Adds an element of chaos                                                

### Your AI
+ Your ride or die companions                                             
+ They will follow your instructions no matter if it cost them their life 
+ Hope you like if statements                                             

#

### Map
+ Each run the map will be randomly generated with walls entrances and of course the objective                        
+ To win you must take control of the place or take all the merchandise before all your team dies
+ The AI will adapt to your strategy over time, use the walls as covers

### Score
+ Each victory will add one point to your score
+ While the game has no END condition there will come a point in which your strat wont work
+ Reach new high scores, create the best strategy
